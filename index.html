Say Cheese!

Minimal javascript library for integrating a webcam and snapshots into your app.

Taking pictures

Say Cheese exposes a simple, event-based API for taking snapshots from your webcam:

{% highlight javascript %} $(function() { var sayCheese = new SayCheese('#container-element'); sayCheese.on('start', function() { this.takeSnapshot(); }); sayCheese.on('snapshot', function(snapshot) { // a snapshot has been taken, do something with it :) }); sayCheese.start(); }); {% endhighlight %}

Compatibility

Works for the following browsers:

  • Google Chrome
  • Opera
  • Firefox with caveats

Firefox users:
Unless you're using the Nightly build, you have to manually enable the webcam functionality. You can do this by browsing to about:config, right clicking the page, and then selecting Boolean in the 'New' menu. Enter media.navigator.enabled, and double click it if it doesn't appear set to true.
